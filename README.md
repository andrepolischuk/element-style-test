# element-style-test

> Test approaches to apply style to element

## Install

```sh
git clone https://github.com/andrepolischuk/element-style-test
cd element-style-test
npm install
```

## Run

```sh
npm start
```

And open [`http://0.0.0.0:3000`](http://0.0.0.0:3000) in your browser.
